# 合唱团成员端小程序 Demo 使用说明

## 1. 导入方式

1. 打开「微信开发者工具」。
2. 选择「导入项目」。
3. 项目目录选择本项目下的 `miniapp` 文件夹。
4. AppID 可先选择「测试号」或使用 `touristappid`。
5. 导入后点击「编译」。

## 2. 后端必须先运行

在项目根目录启动 Docker：

```bash
cd ~/Downloads/choir_app_mvp
docker compose up --build
```

确认后端正常：

```text
http://127.0.0.1:8000/health
```

## 3. 开发者工具本地访问设置

本 Demo 默认接口地址：

```text
http://127.0.0.1:8000
```

在微信开发者工具中，请确认：

- 详情 → 本地设置 → 勾选「不校验合法域名、web-view、TLS 版本以及 HTTPS 证书」

## 4. 真机预览注意事项

如果用手机真机预览，`127.0.0.1` 指的是手机自己，不是电脑。

需要把接口地址改成电脑的局域网 IP，例如：

```text
http://192.168.1.23:8000
```

可在小程序「我的」页面底部修改接口地址并保存。

## 5. 体验流程

### 管理后台

打开：

```text
http://127.0.0.1:8080
```

1. 管理员登录。
2. 创建合唱团，如 `D Major Choir`。
3. 创建作品，如《雪绒花》，并上传 PDF。
4. 创建活动。
5. 创建练习任务。
6. 复制合唱团邀请码。

### 小程序成员端

1. 打开小程序「我的」。
2. 用成员手机号登录，例如 `13900000001`。
3. 输入后台邀请码，申请加入。
4. 回后台「成员审批」通过。
5. 回小程序点击「刷新我的合唱团」。
6. 在首页、活动、任务、谱库中体验成员端流程。

## 6. 当前小程序功能

- 成员登录
- 输入邀请码申请加入合唱团
- 选择当前合唱团
- 查看首页数据
- 查看活动
- 确认参加、请假、签到
- 查看练习任务
- 录音打卡 / Demo 打卡
- 查看本人打卡与点评
- 查看谱库作品与资料
- 打开 PDF / 播放音频

## 7. 当前仍是 Demo 版本

这版小程序适合产品演示和试点验证，还不是正式上线版本。正式上线前还需要：

- 配置正式 AppID
- 配置 HTTPS 域名
- 后端部署到公网服务器
- 文件存储迁移到对象存储
- 完善 UI 细节和异常提示
- 完成隐私政策、用户协议和小程序审核资料
