# 成员端网页版 Demo 使用说明

启动系统：

```bash
docker compose down
docker compose up --build
```

打开：

- 管理后台：http://127.0.0.1:8080
- 成员端网页版：http://127.0.0.1:8090
- 后端 API：http://127.0.0.1:8000/docs

建议演示流程：

1. 在管理后台用 `13800000000 / Admin` 登录。
2. 创建合唱团，并记录邀请码。
3. 在成员端用 `13900000001 / 合唱团成员` 登录。
4. 输入邀请码，申请加入合唱团。
5. 回到管理后台，在“成员审批”通过该成员。
6. 管理后台创建活动、作品、练习任务。
7. 成员端查看活动，确认参加、请假或签到。
8. 成员端查看练习任务，上传录音并提交打卡。
9. 管理后台在“打卡点评”里给成员点评。
10. 成员端查看“我的打卡与点评”和通知。

说明：这是用于演示的浏览器成员端，不替代后续正式微信小程序，但可以帮助你先体验完整前台闭环。
